# Vrinda_Store_Data_Analysis_Project_Excel
##  Project Overview

This project is an **Excel Sales Dashboard** created to analyze the sales performance of **Vrinda Store**. The dashboard transforms raw sales data into meaningful insights using Excel features such as Pivot Tables, Pivot Charts, Slicers, and Interactive Dashboards.

The main objective of this project is to help business owners understand customer behavior, sales trends, order status, and top-performing sales channels for better decision-making.
